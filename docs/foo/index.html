3. Equations in normal blocks

\[k_{n+1} = n^2 + k_n^2 - k_{n-1}\]

Some useful text! Formula for quadratic root:

\$x = \frac{-b \pm \sqrt{b^2 - 4ac}}{2a}\$

Inline equation works too! \(a^2+b^2=c^2\). Or as stem \$a^2+b^2=c^2\$. Pretty nice, huh?
